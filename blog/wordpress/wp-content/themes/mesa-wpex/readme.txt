== Description ==

Mesa WPEX is a clean and simple theme created for blogs. Features a clean and minimal design with grid style archives to display your posts. The theme is built specifically for use with the new WordPress site editor and includes 8 different global color styles to choose from. Use the site editor to fully customize the theme layout.

== Installation ==

1. Make sure you're running WordPress 6.2 or later.
2. Upload the theme.
3. Activate the theme.
4. Go to Appearance > Editor to customize.
